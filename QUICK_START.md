## 📌 Important

### Keep Agent Running
- The `START_AGENT.bat` window must stay open (can be minimized)
- It shows up in the system tray (bottom right)
- Your friends can access the app ONLY while the agent is running

### To Stop the Agent
- Right-click the tray icon (bottom right)
- Click "Exit"

### To Restart the Agent
- Double-click `START_AGENT.bat` again
