================================================================================
              AUDIO SOLUTION - STARTUP STEPS (FOLLOW IN ORDER)
================================================================================

PREREQUISITE: Node.js installed? Run: node --version (should show v16+)

================================================================================

1. Press: Windows Key + R
2. Type:  cmd
3. Press: Enter

    cd "C:\Users\anojs\OneDrive\Desktop\Claude_CLI\Audio Solution\backend"

    npm run dev

    ╚════════════════════════════════════════╝

================================================================================
STEP 6: VERIFY EVERYTHING WORKS
================================================================================

On the dashboard, you should see:

================================================================================
STEP 7: TEST CREATE ROUTE
================================================================================

1. Click "Spotify" in the Audio Sources list
2. Click "Speakers" in the Output Devices list
3. Click "Create Route"
4. Look at "Active Routes" - you should see your new route
5. Click the delete icon to remove the route

If this works, everything is functioning! ✅

================================================================================
WHAT IF SOMETHING GOES WRONG?
================================================================================

Backend won't start?
→ Port 5000 in use. Run:
   netstat -ano | findstr :5000
   taskkill /PID XXXX /F

Frontend shows blank page?
→ Press Ctrl+Shift+Delete to clear cache
→ Or open in Incognito window
→ Or try Ctrl+R to refresh

Can't login?
→ Make sure you registered first
→ Check email spelling matches
→ Open browser console (F12) for errors

One terminal closed by accident?
→ Just restart that terminal with the npm run dev command
→ The app will reconnect automatically

================================================================================
STOPPING THE APPLICATION
================================================================================

When done, press Ctrl+C in each terminal window (in any order):
- Terminal 1 (Backend)
- Terminal 2 (Frontend)
- Terminal 3 (Audio Agent)

All servers will stop cleanly.
